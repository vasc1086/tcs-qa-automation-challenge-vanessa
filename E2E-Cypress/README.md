# E2E Cypress - Purchase Testing

# Ejercicio 1 – Prueba E2E con Cypress (Demoblaze)

Este módulo contiene la automatización del flujo de compra en la página https://www.demoblaze.com/ utilizando Cypress.

## ✔ Flujo automatizado
1. Ingresar al sitio Demoblaze
2. Agregar dos productos al carrito
3. Visualizar el carrito
4. Completar el formulario de compra
5. Finalizar compra exitosamente

## ✔ Requisitos
- Node.js 16+
- Cypress 12+
