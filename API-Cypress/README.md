# API Cypress - Authentication Testing

# Ejercicio 2 – Pruebas de API con Cypress

Este módulo contiene pruebas automatizadas para los endpoints Signup y Login del servicio de Demoblaze.

## ✔ Endpoints probados
- POST https://api.demoblaze.com/signup
- POST https://api.demoblaze.com/login

## ✔ Escenarios implementados
1. Crear usuario nuevo
2. Crear usuario existente
3. Login exitoso
4. Login fallido

## ✔ Requisitos
- Node.js 16+
- Cypress 12+
